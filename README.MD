A comprehensive portfolio showcasing three professional data analytics case studies, deployed on GitHub Pages with interactive visualizations and professional presentation.

## 🎯 Portfolio Overview

This repository contains a complete data analytics portfolio featuring three distinct case studies that demonstrate expertise in data analysis, visualization, and storytelling across different domains and tools.

## 🚀 Case Studies

### 1. NYC Citibike Analysis 2022 🚴‍♀️
- **File**: `Citibike_Enhanced_Case_Study.html`
- **Description**: Comprehensive analysis of NYC Citibike usage patterns throughout 2022
- **Technologies**: Python, Pandas, Matplotlib, Seaborn, Folium
- **Features**: Interactive maps, seasonal trends, station analysis, user behavior insights
- **Key Insights**: Peak usage patterns, popular routes, seasonal variations

### 2. Flavours of Cacao Analysis 🍫
- **File**: `Cacao_Analysis_Portfolio.html`
- **Description**: Interactive Tableau dashboard exploring global chocolate ratings and origins
- **Technologies**: Tableau, Data Visualization, Statistical Analysis
- **Features**: Geographic mapping, quality analysis, flavor profiling
- **Key Insights**: Regional quality patterns, rating distributions, premium chocolate characteristics

### 3. Data Storytelling Showcase 📊
- **File**: `Exercise2_9_Storytelling_Visualization.html`
- **Description**: Advanced Tableau story demonstrating narrative techniques through data
- **Technologies**: Tableau, Interactive Design, UX/UI
- **Features**: Story-driven visualization, interactive elements, professional presentation
- **Key Insights**: Effective data storytelling methodologies and user engagement

## 📋 Deployment Instructions

### Initial Setup
1. Clone or fork this repository
2. Ensure GitHub Pages is enabled in repository settings
3. Set source to "Deploy from a branch" → main branch → / (root)

### File Management
- `index.html` serves as the main homepage (required for GitHub Pages)
- All HTML files are self-contained with embedded CSS and JavaScript
- No additional dependencies required for deployment

### Local Development
```bash
# Clone the repository
git clone https://github.com/ayshashanaz/Case-Studies-2025.git

# Navigate to directory
cd Case-Studies-2025

# Open any HTML file in browser for local viewing
open index.html
```
